# 🏥 Hastane Kayıt Sistemi (HKS)
Bu proje, Java Spring Boot ve MSSQL kullanılarak geliştirilmiş modern bir randevu otomasyonudur.

### 🛠 Teknolojiler
- **Backend:** Java, Spring Boot, Spring Data JPA
- **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript (Fetch API)
- **Database:** Microsoft SQL Server (MSSQL)

### 🚀 Özellikler
- TC Kimlik No ve Şifre ile güvenli giriş.
- Şehir, İlçe ve Hastane hiyerarşisine göre doktor seçimi.
- Randevu oluşturma ve geçmiş randevuları iptal edebilme.
